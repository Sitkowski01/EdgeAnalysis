Instrukcje uruchomienia projektu:

1. Zainstaluj zależności:

   npm install

2. Uruchom serwer deweloperski:

   npm run dev

3. Otwórz http://localhost:5173 (lub adres podany przez Vite)

Projekt używa Vite + React + TypeScript + Tailwind. Motyw (ciemny/jasny) zapisuje się w localStorage pod kluczem `ea-theme`.
